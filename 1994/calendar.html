One page calendar

Projeto inspirado em um artigo da revista BigThink sobre um modelo de calendário de 1 página. Site Repositório

Objetivo

Tenho há alguns anos interesse por calendários, principalmente a visualização do calendário em uma única página. Algo bem específico, eu sei XD. Antigamente eu criava um calendário anual de forma manual e com colagens , e recentemente começei a realizar esse processo de criação no figma.

Após ler este artigo artigo do BigThink sobre um calendário de uma página que pode ser usado por vários anos, basta mudar a tabela do mês para cada ano. Achei a ideia super interessante a ideia e achei e quando estava praticando React durante o curso "Joy of React" achei que um dos projeto poderia ser um site baseado na ideia do one page calendar.

Desafios

No geral diria que foi um desafio organizar o código, o contexto de cada customização, penso ainda em refatorar o código e melhora-lo.

Inicialmente queria simplesmente usar o input de cor padrão do HTML porém acabei no percurso do projeto criando meu próprio componente. Não criei um colorwheel, decide por algo mais simples mas mesmo assim levou trabalho pensar no que seria melhor para o uso. Comecei o componente com RGBA e depois passei para HSLA, e no layout usei css grid.

Também fui desafiada com o css devido a quantidade de customização, tive que planejar e organizar bem o css para que tudo desse certo.

Funcionalidades

  • Exibir o calendário do ano desejado em uma visão de grid como no artigo;
  • Customização do calendário e dados salvos no local storage;
  • Salvar e excluir eventos no calendário e ter a opção de exportar dados como .csv

Tech Stack

  • Vite
  • React
  • Javascript
  • RadixUI
  • React Draggable
