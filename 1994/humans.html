Humans of Seoul

Projeto criado para exibir o RSS Feed do site Humans of Seoul com somente o texto original coreano visível e a tradução disponível se desejado. Site Repositório

Objetivos

Tenho interesse em várias linguas e recentemente tenho começado a aprender coreano. Por isso para praticar o idioma comecei a procurar contéudo online em coreano e com tradução e encontrei o site "Humans of Seoul". Este projeto é baseado no projeto "Humans of Nova York", projeto que realiza pequenas entrevistas com pessoas da cidade.

No site do projeto está disponibilizado as entrevistas em coreano e logo abaixo a tradução em inglês. Como o site possui um feed RSS pensei que poderia formata-lo para ler somente o original coreano para praticar minha leitura no idioma e assim só vializar a tradução depois para confirmar meu entendimento.

Funcionalidades

  • Customizações das cores e tipografia;
    • Mudança de tema;
    • Tamanho, espessura e tipo de tipografia;
  • Espaçamento de palavras e linha;
  • Alternar a visibilidade da tradução por linha;
  • Ao passar o mouse por uma palavra exibir popover com link para o dicionário Papago;
  • Salvar customizações e palavras selecionadas no Local Storage.

Desafios

Não havia ainda tido oportunidade de trabalhar com dados em XML, que é formato que o RSS Feed é fornecido. Então foi um processo de pesquisa para entender como fazer o parse e formatar. Também tive que interar na lógica para separar o texto em coreano e o texto traduzido, inicialmente estava usando o layout mas como essa lógica estava propensa a falhar ajustei o código para utilizar da biblioteca Franc para realizar a detecção do idioma.

Tech Stack

  • Vite
  • React
  • Typescript
  • FRANC - Natural language detection
  • SWR - React Hooks para Data Fetching
  • RadixUI
  • Lucide React
